1. unzip the file corresponding to your system:

eg:
```
unzip snoke-binary-ubuntu-latest-4.13.x.zip
```

This should give you the `snoke.tar` archive.

2. move it to your favorite dir (`$HOME`, or `/usr/local/`, or whatever)

3. untar

```
tar xvf snoke.tar 
```

4. execute `bin/snoke`
